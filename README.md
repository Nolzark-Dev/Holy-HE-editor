## Installation

1. Clone the repository and cd into it
```
git clone https://github.com/Nolzark-Dev/Holy-HI-editor.git
cd Holy-HI-editor
```
2. Give install.sh execute permissions
```
chmod +x install.sh
```
3. Run the installer then have fun :)
```
./install.sh
```
